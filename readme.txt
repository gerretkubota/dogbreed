========================
Purpose of program.
========================
- Purpose
	- Table view of all the dog breed list
	- API call from dog.ceo API to extract data
=========================
How to RUN program.
=========================
- Prerequisite
  	- Make sure to have npm and node installed on machine.
	- npm start at the root of the project’s directory in a command line/terminal.

=========================
How to USE program.
=========================
- Click on the type of breed to have another row display with the image of the breed that is clicked on. Click again to close.
